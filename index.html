Grammar Analysis

Grammar Rules

S → AB
A → aA | ε
B → bB | ε
FIRST Sets
FIRST(S) = {a, ε}
FIRST(A) = {a, ε}
FIRST(B) = {b, ε}
The FIRST set of a symbol contains all terminals that can appear as the first symbol of any string derived from that symbol.
FOLLOW Sets
FOLLOW(S) = {$}
FOLLOW(A) = {b}
FOLLOW(B) = {$}
The FOLLOW set of a non-terminal contains all terminals that can appear immediately to the right of that non-terminal in any valid derivation.
